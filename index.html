ChatterBox

Overview:
A stripped down version of slack, with user login featuring realtime chat interaction and a user list displaying active users. Users and messages stored in a Postgres database.

Struggles:
Figuring out whether or not to inject direct Npgsql or use an ORM such as Entity Framework to map out my database structure. I applied both, but went with direct Npgsql injection.

 C#
 ReactJs
 ASP.NET
 PostgreSQL

Twitter Showcase

Overview:
A WebAPI app designed to search for tweets using keywords and also a showcase of 5 of my favorite Twitter users that displays a random tweet from them with a single click.

Struggles:
I came across the daunting CORS issue but quickly overcame it by pulling the necessary API information to the backend and routing the JSON information to my frontend. Creating a relationship via Axios GET methods from the frontend to the backend.

 ReactJs
 NodeJs
 Heroku
